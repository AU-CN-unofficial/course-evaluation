# 奥胡斯大学课程评价

奥胡斯大学（CS系）有很多很棒的课，也有一些无聊的课，或者考核不合理的课。
我们希望前人走过的弯路，后人不要再趟；希望热爱知识的同学不会错过一些精彩的课；
希望付出了努力的同学不会在成绩上留下遗憾……
